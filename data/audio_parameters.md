## 一、压缩器设置
### 人声温和压缩
**描述**：适用于对话、歌唱、播客的人声温和压缩
**适用场景**：vocal_recording
**难度等级**：beginner
**参数配置**：
- 阈值：`-20 dB` → 保持自然动态，不过度压缩
- 比率：`2.5:1` → 温和压缩，保留动态变化
- 启动时间：`15 ms` → 保留唇齿音瞬态，避免过度压缩起始部分
- 释放时间：`200 ms` → 平滑呼吸起伏，自然过渡
- 增益补偿：`3 dB` → 补偿压缩损失的电平
**适用场景**：对话录音、歌唱录音、播客人声
**注意事项**：避免过度压缩导致呼吸声放大，避免过快启动时间损失瞬态

---
### 鼓组并行压缩
**描述**：鼓组和打击乐器的并行压缩，保留动态同时增加密度
**适用场景**：drum_processing
**难度等级**：intermediate
**参数配置**：
- 阈值：`-15 dB` → 仅压缩峰值，不过度压缩
- 比率：`4:1` → 中等压缩力度
- 启动时间：`8 ms` → 控制瞬态，避免鼓声过冲
- 释放时间：`80 ms` → 快速恢复，适合快速节奏
- 混合比例：`30%` → 30%压缩信号 + 70%干声，保留动态
**适用场景**：鼓组处理、打击乐器、节奏乐器
**注意事项**：避免100%压缩信号损失动态，避免过慢释放时间导致 pumping 效应

---
### 总线压缩
**描述**：总线和母带处理的透明压缩
**适用场景**：mastering
**难度等级**：advanced
**参数配置**：
- 阈值：`-10 dB` → 轻微控制，不过度压缩
- 比率：`1.5:1` → 透明压缩，几乎不可察觉
- 启动时间：`30 ms` → 不压缩瞬态，保留冲击力
- 释放时间：`auto` → 自动适应节奏
- 增益补偿：`1 dB` → 轻微增益补偿
**适用场景**：总线处理、母带处理、立体声总线
**注意事项**：避免过度压缩损失动态，避免过快启动时间压扁瞬态

---
### 贝斯温和压缩
**描述**：适用于电贝斯和合成贝斯的温和压缩，保持动态同时控制峰值
**适用场景**：bass_processing
**难度等级**：intermediate
**参数配置**：
- 阈值：`-18 dB` → 控制峰值，保持自然动态
- 比率：`3:1` → 温和压缩，保持演奏动态
- 启动时间：`20 ms` → 保留拨弦瞬态，避免过度压缩
- 释放时间：`150 ms` → 平滑音符衰减，保持连贯性
- 增益补偿：`2 dB` → 补偿压缩损失的电平
**适用场景**：电贝斯、合成贝斯、低音乐器
**注意事项**：避免过快启动时间损失拨弦冲击力，避免过慢释放时间导致 pumping 效应

---
### 人声强力压缩
**描述**：适用于摇滚、说唱等需要高密度人声的强力压缩
**适用场景**：vocal_processing
**难度等级**：advanced
**参数配置**：
- 阈值：`-25 dB` → 深度压缩，创造高密度人声
- 比率：`6:1` → 高比率压缩，大幅压缩动态范围
- 启动时间：`5 ms` → 快速启动，控制瞬态峰值
- 释放时间：`100 ms` → 中等释放时间，保持声音紧凑
- 增益补偿：`6 dB` → 大幅增益补偿，提升整体电平
**适用场景**：摇滚人声、说唱人声、高冲击力人声
**注意事项**：避免过度压缩导致呼吸声放大，避免过慢启动时间导致瞬态过冲

## 二、均衡器设置
### 男声均衡
**描述**：男声频率平衡处理
**适用乐器**：male_vocal
**难度等级**：beginner
**参数配置**：
1. 高通滤波：`80 Hz` 切除 → 切除 rumble 和低频噪音
2. 200 Hz 钟型：`+2 dB, Q=1.0` 提升 → 增加丰满度和温暖感
3. 400 Hz 钟型：`-3 dB, Q=1.5` 衰减 → 减少浑浊感，增加清晰度
4. 3000 Hz 钟型：`+4 dB, Q=1.0` 提升 → 增强清晰度和临场感
5. 10000 Hz 钟型：`+2 dB, Q=0.7` 提升 → 增加空气感和开放感
**适用场景**：男声录音、播客、旁白
**注意事项**：根据具体嗓音调整频率，避免过度提升导致刺耳

---

4. 5000 Hz 钟型：`+3 dB, Q=1.5` 提升 → 增强临场感和细节
**适用场景**：摇滚吉他、金属吉他、独奏吉他
**注意事项**：根据吉他音箱和拾音器类型调整频率

## 三、混响设置
### 板式混响
**描述**：适用于人声和乐器的板式混响
**混响类型**：plate
**难度等级**：beginner
**参数配置**：
- 预延迟：`20 ms`（推荐范围15-25ms）→ 避免与原声重叠，增加清晰度
- 衰减时间：`1.5 s`（推荐范围1.2-1.8s）→ 适合人声的衰减时间
- 高频切除：`0.8 factor`（推荐范围0.7-0.9）→ 模拟高频吸收，避免刺耳
- 干湿比：`20%`（推荐范围15-25%）→ 干湿混合比例
**适用场景**：人声、弦乐、钢琴
**注意事项**：板式混响适合增加密度和融合度

---
### 大厅混响
**描述**：大型空间的大厅混响
**混响类型**：hall
**难度等级**：intermediate
**参数配置**：
- 空间大小：large
- 早期反射：`40 ms`（推荐范围30-50ms）
- 衰减时间：`2.5 s`（推荐范围2.0-3.0s）
- 扩散度：`0.8 factor`（推荐范围0.7-0.9）
- 干湿比：`25%`（推荐范围20-30%）
**适用场景**：交响乐、影视配乐、管弦乐
**注意事项**：大厅混响适合创造宏大空间感

---
### 房间混响
**描述**：中小型房间的自然混响
**混响类型**：room
**难度等级**：beginner
**参数配置**：
- 空间大小：medium
- 预延迟：`15 ms`（推荐范围10-20ms）
- 衰减时间：`1.0 s`（推荐范围0.8-1.2s）
- 扩散度：`0.7 factor`（推荐范围0.6-0.8）
- 干湿比：`15%`（推荐范围10-20%）
**适用场景**：鼓组、钢琴、室内乐
**注意事项**：房间混响适合增加真实感和融合度，不改变原始音色

---
### 弹簧混响
**描述**：经典吉他音箱弹簧混响效果
**混响类型**：spring
**难度等级**：intermediate
**参数配置**：
- 弹簧数量：2
- 衰减时间：`2.5 s`（推荐范围2.0-3.0s）
- 阻尼：`0.6 factor`（推荐范围0.5-0.7）
- 音色特性：vintage
- 干湿比：`30%`（推荐范围25-35%）
**适用场景**：电吉他、复古音色、Lo-fi音乐
**注意事项**：弹簧混响有独特的金属感共鸣，适合复古和实验性音色

## 四、动态处理器
### 人声齿音消除
**描述**：消除人声齿音（s, sh, ch等）
**设备类型**：deesser
**难度等级**：beginner
**参数配置**：
- 工作频率：`6000 Hz`（推荐范围5000-7000Hz）
- 阈值：`-30 dB`（推荐范围-35~-25dB）
- 比率：`3:1`（推荐范围2:1~4:1）
- 带宽：narrow
- 侧链模式：启用
**适用场景**：人声处理、语音录音、歌唱处理
**注意事项**：监听侧链信号，确保只压缩齿音频率

---
### 母带限制器
**描述**：母带处理最终限制器
**设备类型**：limiter
**难度等级**：advanced
**参数配置**：
- 峰值上限：`-1.0 dB TP` → True Peak天花板，防止 clipping
- 阈值：`-3 dB` → 限制阈值
- 释放时间：`100 ms` → 释放时间
- 过采样：4x
**适用场景**：母带处理、流媒体优化、响度标准化
**注意事项**：使用True Peak限制防止 intersample peaks

---
### 多段压缩器
**描述**：分频段独立压缩，针对不同频率范围进行精确控制
**设备类型**：multiband_compressor
**难度等级**：advanced
**参数配置**：
- 低频段：分频点150Hz，阈值-18dB，比率3:1
- 中频段：分频点1500Hz，阈值-15dB，比率2.5:1
- 高频段：分频点5000Hz，阈值-12dB，比率2:1
- 交叉滤波器：steep
- 增益补偿：auto
**适用场景**：母带处理、复杂混音、动态修复
**注意事项**：多段压缩适合解决特定频率范围的动态问题，避免整体压缩带来的副作用

---
### 噪声门
**描述**：低于阈值的信号静音，用于消除背景噪音和串音
**设备类型**：gate
**难度等级**：intermediate
**参数配置**：
- 阈值：`-40 dB`（推荐范围-45~-35dB）
- 启动时间：`5 ms`（推荐范围1-10ms）
- 保持时间：`50 ms`（推荐范围20-100ms）
- 释放时间：`100 ms`（推荐范围50-200ms）
- 衰减范围：`60 dB` → 门关闭时的衰减量
**适用场景**：鼓组处理、语音清理、录音降噪
**注意事项**：设置合适的保持和释放时间，避免门频繁开关导致的 chopping 效果

---
### 扩展器
**描述**：低于阈值的信号进一步衰减，增加动态范围
**设备类型**：expander
**难度等级**：advanced
**参数配置**：
- 阈值：`-30 dB`（推荐范围-35~-25dB）
- 比率：`1.5:1`（推荐范围1:1.2~1:2）
- 启动时间：`10 ms`（推荐范围5-20ms）
- 释放时间：`150 ms`（推荐范围100-300ms）
**适用场景**：增加动态范围、背景噪音抑制、混音清晰度提升
**注意事项**：扩展器相比噪声门更平滑，适合需要自然过渡的场景
