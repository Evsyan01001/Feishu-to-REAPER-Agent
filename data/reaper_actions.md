# REAPER 远程控制指令集 (Action ID Map)

> **文档说明**：本表定义了飞书 Agent 控制 REAPER 的核心 Action ID。当用户下达操作指令时，Agent 应通过此表检索对应的 ID。

## 1. 播放与导航 (Transport & Navigation)
| Action ID | 功能描述 | 语义关键词 (触发词) |
| :--- | :--- | :--- |
| **40001** | 播放 / 暂停切换 | 播放, 暂停, 开始听, 继续, play, pause |
| **40002** | 停止播放 | 停止, 结束播放, stop |
| **40003** | 返回项目起始点 | 回到开头, 重头开始, rewind, start of project |
| **40004** | 跳转到上一个标记点 | 上一个标记, 上一个 Marker, previous marker |
| **40005** | 跳转到下一个标记点 | 下一个标记, 下一个 Marker, next marker |
| **40046** | 循环播放开关 | 循环播放, 开启循环, loop on/off |
