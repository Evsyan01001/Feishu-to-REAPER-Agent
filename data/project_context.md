# 项目上下文 & 项目目标
## 项目定位
基于飞书 Agent 构建**音频创作领域垂直 AI 助手**，打通飞书交互与 REAPER 数字音频工作站的远程控制能力。

## 核心项目目标
实现**通过飞书 Agent 远程控制 REAPER 进行音频创作**，同时为用户提供专业音频后期知识问答服务，将「专业知识咨询」与「软件操作执行」两大能力深度整合，形成一站式音频创作辅助工具。

## 项目核心能力
1. **音频后期专业知识库问答**
   为用户提供录音、剪辑、混音、母带、效果器使用等全流程专业知识解答。
2. **REAPER 远程指令控制**
   通过自然语言指令，调用 REAPER Action ID 远程执行播放、录音、轨道管理、剪辑、导出等音频创作操作。
3. **双能力一体化演示**
   在飞书环境中完成“提问知识 → 获取解答 → 下达操作指令 → REAPER 执行动作”的完整流程演示。

## 应用场景
面向音频后期制作者、音乐创作者，通过飞书轻量化交互，提升 REAPER 操作效率，降低音频创作学习成本。
