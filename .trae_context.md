# Feishu Agent 项目上下文
## 项目基础信息
项目名称：Feishu Agent - 智能音频助手
技术栈：Python 3.8+, Flask, ChromaDB, Redis, DeepSeek API
核心特色：飞书机器人 + RAG专业知识库 + REAPER音频工作站自然语言控制
当前版本：v1.0.0
开源地址：https://github.com/Evsyan01001/Feishu-to-REAPER-Agent

## 项目定位
基于 Agent 构建**音频创作领域垂直 AI 助手**，打通飞书交互与 REAPER 数字音频工作站的远程控制能力。

## 核心项目目标
实现**通过飞书 Agent 远程控制 REAPER 进行音频创作**，同时为用户提供专业音频后期知识问答服务，将「专业知识咨询」与「软件操作执行」两大能力深度整合，形成一站式音频创作辅助工具。

## 项目核心能力
1. **音频后期专业知识库问答**
   为用户提供录音、剪辑、混音、母带、效果器使用等全流程专业知识解答。
2. **REAPER 远程指令控制**
   通过自然语言指令，调用 REAPER Action ID 远程执行播放、录音、轨道管理、剪辑、导出等音频创作操作。
3. **双能力一体化演示**
   在飞书环境中完成“提问知识 → 获取解答 → 下达操作指令 → REAPER 执行动作”的完整流程演示。

## 应用场景
面向音频后期制作者、音乐创作者，通过飞书轻量化交互，提升 REAPER 操作效率，降低音频创作学习成本。

## 系统架构总览
1. 用户交互层：支持CLI终端、飞书App、HTTP API三种接入方式
2. 应用服务层：FeishuAgent主控制器负责消息路由、会话管理、指令分发
3. REAPER控制层：实现自然语言转REAPER操作，通过文件系统与REAPER端Lua脚本通信
4. 智能处理层：RAG向量检索、DeepSeek API对话、多轮会话管理
5. 数据存储层：ChromaDB向量库、Redis缓存、本地文件系统
